# DBD Strategy Board

Live: https://dogdbd.github.io/dbd-strategy-board/

A simple web tool for planning Dead by Daylight maps:
import a map image → click to place markers → drag to adjust.

## How to use
1. Open the site.
2. Get a map image (PNG/WebP), for example from Hens' Callouts: https://hens333.com/callouts  
   - Tip: click a map image there to focus it, then save/download it.
3. In DBD Strategy Board, click **Import map** and select the image file.
4. Place and edit markers (Generator / Killer / Survivor).
5. Optional sharing:
   - **Export Shareable HTML**: creates a single `.html` file that *embeds the map image* + your markers, so others can open it in a browser without needing the map image separately.
   - **Export JSON**: saves markers + view settings only (the map image is **not** embedded), so others need the same map file name/image to use it.
